Política de Privacidade do Bot MeuBot

Última atualização: 22/05/2025

Dados que coletamos

  • IDs de usuário, servidor, canal ou mensagem, quando necessário para o funcionamento do bot.
  • Preferências e configurações feitas pelo usuário dentro do bot.
  • Logs de comandos usados (sem conteúdo pessoal).

Como usamos os dados

  • Para responder a comandos e prover funcionalidades.
  • Para manter o bot operando corretamente e com segurança.

O que não fazemos

  • Não compartilhamos seus dados com terceiros.
  • Não vendemos informações.
  • Não usamos dados para fins de marketing.

Armazenamento

Todos os dados são armazenados de forma segura e são apagados mediante solicitação do usuário ou remoção do bot do servidor.
